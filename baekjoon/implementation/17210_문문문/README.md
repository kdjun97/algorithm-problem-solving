# 백준 17210번 - 문문문

# 풀이

이 문제는 규칙이 있다.  

규칙 2번이 연속으로 같은 방법으로 문을 열 수 없음.  
-> 여기서 첫 번째 수가 굉장히 중요해진다.  
첫 번째가 0이면 0 1 0 1 0 1 이런식으로 문을 열어야 하고,  
1이라면 1 0 1 0 1 0 이런식으로 문을 열어야 한다.  

규칙 3에서 2의 배수 문은 동일한 방식으로 열어야 함.  
규칙 4에서 3의 배수 문 역시 동일한 방식으로 열어야 함.  

여기서 규칙 4를 보면 어떤 형태로 첫 문을 열어도 달성이 되지 않는다.  
따라서, 6번째 이상의 문은 모두 탈출 불가능이다.  
이 점을 이용하여 풀었다.
